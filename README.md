## Installation instruction
1. Copy the project:
```bash
git clone https://github.com/rosiaAI/Bivan_makeup
```
2. Go to your project folder:
```bash
cd Bivan_makeup
```
3. Install lib:
```bash
pip install flask
```
4. Start the project:
```bash
python app.py
```
